еҹәжң¬д»Ӣз»ҚВ¶

JSON SchemaдёҖдёӘз”ЁжқҘе®ҡд№үе’Ңж ЎйӘҢJSONзҡ„Webи§„иҢғгҖӮз®ҖиҖҢиЁҖд№ӢпјҢJSON SchemaжҳҜз”ЁжқҘж ЎйӘҢJSONжҳҜеҗҰз¬ҰеҗҲйў„жңҹ

JSON Schemaж”ҜжҢҒжЈҖжөӢзҡ„зұ»еһӢВ¶

еңЁJSON SchemaдёӯпјҢеҸҜд»ҘжЈҖжөӢеҰӮдёӢзұ»еһӢпјҡ

зұ»еһӢ и§ЈйҮҠ
string еӯ—з¬ҰдёІзұ»еһӢпјҢз”ЁдәҺж–Үжң¬ж•°жҚ®
number ж•°еӯ—зұ»еһӢпјҢз”ЁдәҺиЎЁзӨәжө®зӮ№ж•°
integer ж•ҙж•°зұ»еһӢпјҢз”ЁдәҺиЎЁзӨәж•ҙж•°
boolean еёғе°”зұ»еһӢпјҢеҖјдёәtrueжҲ–false
object еҜ№иұЎзұ»еһӢпјҢз”ЁдәҺеөҢеҘ—зҡ„JSONеҜ№иұЎ
array ж•°з»„зұ»еһӢпјҢз”ЁдәҺеҲ—иЎЁжҲ–йӣҶеҗҲ
null з©әеҖјзұ»еһӢ

йңҖиҰҒж ЎйӘҢжҹҗдёҖдёӘеҜ№иұЎд№Ӣдёӯзҡ„еұһжҖ§зұ»еһӢеҸҜд»ҘдҪҝз”Ё"type"иҝӣиЎҢзұ»еһӢжҢҮе®ҡпјҢдҫӢеҰӮпјҡ

жңҖеӨ§еҖје’ҢжңҖе°ҸеҖјВ¶

еңЁJSON SchemaдёӯпјҢжңүдёӨз§Қж–№ејҸиЎЁзӨәжңҖеӨ§еҖје’ҢжңҖе°ҸеҖјпјҡ

  1. maximumе’ҢminimumпјҡжҢҮе®ҡзұ»еһӢзҡ„жңҖеӨ§еҖје’ҢжңҖе°ҸеҖјпјҲеҢ…еҗ«maximumе’ҢminimumеҖјпјү
  2. exclusiveMaximumе’ҢexclusiveMinimumпјҡжҢҮе®ҡзұ»еһӢзҡ„жңҖеӨ§еҖје’ҢжңҖе°ҸеҖјпјҲдёҚеҢ…еҗ«exclusiveMinimumе’ҢexclusiveMaximumеҖјпјү

еӯ—з¬ҰдёІж ЎйӘҢВ¶

еүҚйқўжҸҗеҲ°дәҶй’ҲеҜ№зұ»еһӢзҡ„ж ЎйӘҢпјҢдҪҶжҳҜдҪҝз”Ё"type": "string"еҸӘиғҪжЈҖжөӢжҳҜеҗҰжҳҜеӯ—з¬ҰдёІзұ»еһӢпјҢдёҚиғҪеҲӨж–ӯеӯ—з¬ҰдёІе…·дҪ“зҡ„еҶ…е®№жҳҜеҗҰз¬ҰеҗҲйңҖжұӮгҖӮдёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҢеңЁJSON Schemaдёӯж”ҜжҢҒдҪҝз”ЁжӯЈеҲҷиЎЁиҫҫејҸеҜ№еӯ—з¬ҰдёІиҝӣиЎҢж ЎйӘҢпјҢеҜ№еә”зҡ„й”®дёәpatternгҖӮе…ідәҺжӯЈеҲҷиЎЁиҫҫејҸзҡ„иҜӯжі•д»Ӣз»Қи§Ғе…ідәҺжӯЈеҲҷиЎЁиҫҫејҸ
