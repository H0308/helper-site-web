  3. зӯүеҫ…пјҲwaitingпјүпјҡиЎЁзӨәиҝӣзЁӢжӯЈеңЁзӯүеҫ…е…·дҪ“дәӢд»¶еҸ‘з”ҹпјҢд№ҹиў«з§°дёәйҳ»еЎһзҠ¶жҖҒ
  4. е°ұз»ӘпјҲreadyпјүпјҡзӯүеҫ…иў«и°ғеәҰеҷЁи°ғеәҰжү§иЎҢ
  5. з»ҲжӯўпјҲterminatedпјүпјҡиҝӣзЁӢе®ҢжҲҗжү§иЎҢ

жү§иЎҢиҝҮзЁӢеҰӮдёӢпјҡ

  • еҪ“иҝӣзЁӢеҲӣе»әжҲҗеҠҹеҗҺпјҲnewпјүпјҢе…¶зҠ¶жҖҒиҪ¬еҢ–дёәе°ұз»ӘпјҲreadyпјүпјҢзӯүеҫ…и°ғеәҰеҷЁи°ғеәҰпјҲscheduler dispatchпјүпјҢи°ғеәҰеҲ°еҪ“еүҚиҝӣзЁӢеҗҺејҖе§ӢиҝҗиЎҢпјҲrunningпјүпјҢзЁӢеәҸжӯЈеёёз»“жқҹйҖҖеҮәпјҲexitпјүеҗ‘ж“ҚдҪңзі»з»ҹиҝ”еӣһж•°жҚ®пјҢжңҖеҗҺз»ҲжӯўпјҲterminatedпјү
  • ж•ҙдёӘиҝҮзЁӢдёӯж¶үеҸҠеҲ°зӯүеҫ…е’Ңдёӯж–ӯпјҢдҫӢеҰӮпјҡеҪ“зЁӢеәҸйңҖиҰҒиҝӣиЎҢзұ»дјјдәҺIOжҲ–иҖ…е…¶д»–дәӢд»¶пјҲI/O or event waitпјүж—¶е°ұдјҡиҝӣе…Ҙзӯүеҫ…зҠ¶жҖҒпјҢзӯүеҫ…IOз»“жқҹжҲ–иҖ…е…¶д»–дәӢд»¶з»“жқҹпјҲI/O or event completionпјүеҶҚд»Һзӯүеҫ…иҪ¬жҚўдёәе°ұз»ӘзҠ¶жҖҒпјҲreadyпјүзӯүеҫ…и°ғеәҰеҷЁи°ғеәҰйҮҚж–°иҝӣе…ҘиҝҗиЎҢзҠ¶жҖҒпјҲrunningпјү

зӯүеҫ…зҠ¶жҖҒзҡ„жң¬иҙЁ¶

дёӢйқўйҮҚзӮ№иҖғиҷ‘зӯүеҫ…пјҲwaitingпјүзҠ¶жҖҒ

иҝӣзЁӢеңЁиў«еҲӣе»әд№ӢеҗҺпјҢжӯӨж—¶ж №жҚ®ж“ҚдҪңзі»з»ҹгҖҢе…ҲжҸҸиҝ°пјҢеҶҚз»„з»ҮгҖҚзҡ„з®ЎзҗҶж–№ејҸпјҢеңЁеҲӣе»әиҝӣзЁӢж—¶пјҢдјҡеҪўжҲҗеҜ№еә”иҝӣзЁӢзҡ„PCBпјҲдҫӢеҰӮLinuxдёӢзҡ„task_structпјүпјҢжӯӨж—¶гҖҢжҸҸиҝ°гҖҚе·Із»Ҹе®ҢжҲҗ

жҺҘзқҖзЁӢеәҸиҝӣе…Ҙе°ұз»ӘзҠ¶жҖҒпјҢжӯӨж—¶ж“ҚдҪңзі»з»ҹдјҡе°ҶиҝӣзЁӢеҜ№еә”зҡ„PCBеҠ иҪҪеҲ°е°ұз»ӘйҳҹеҲ—дёӯпјҢеңЁLinuxдёӢжҳҜдёҖиҲ¬жҳҜдҪҝз”ЁеҸҢеҗ‘й“ҫиЎЁз»“жһ„еҜ№жҜҸдёҖдёӘPCBиҝӣиЎҢиҝһжҺҘпјҢзӨәж„ҸеӣҫеҰӮдёӢпјҢе…¶дёӯcurrentжҢҮй’ҲиЎЁзӨәеҪ“еүҚжӯЈеңЁиў«жү§иЎҢзҡ„иҝӣзЁӢпјҡ

жҜҸдёҖдёӘCPUйңҖиҰҒжү§иЎҢиҝӣзЁӢпјҢе°ұйңҖиҰҒдёҖдёӘдёҺе°ұз»ӘйҳҹеҲ—жңүе…ізҡ„з»“жһ„пјҢиҝҷдёӘз»“жһ„еңЁLinuxдёӢиў«з§°дёәиҝҗиЎҢйҳҹеҲ—пјҲе…·дҪ“и§Ғи°ғеәҰз®—жі•йғЁеҲҶпјүпјҢдёҖиҲ¬иҜҘз»“жһ„дёӯдјҡеӯҳеңЁдёҖдёӘжҢҮй’ҲпјҢиҜҘжҢҮй’ҲжҢҮеҗ‘жӯЈеңЁиҝҗиЎҢзҡ„иҝӣзЁӢпјҢдҫӢеҰӮдёӢйқўзҡ„з®ҖеҢ–еӣҫдёӯзҡ„headжҢҮй’ҲжҢҮеҗ‘зҡ„е°ұжҳҜеҪ“еүҚиҝӣзЁӢпјҢиҖҢtailиЎЁзӨәеҪ“еүҚй—ІзҪ®пјҲidleпјүзҡ„иҝӣзЁӢ

еҰӮжһңжӯӨж—¶зЁӢеәҸйңҖиҰҒиҝӣиЎҢI/Oж“ҚдҪңпјҢеӣ дёәI/Oж“ҚдҪңйҖҹеәҰиҝңе°ҸдәҺCPUзҡ„жү§иЎҢйҖҹеәҰпјҢеңЁеҲҶж—¶ж“ҚдҪңзі»з»ҹдёӯпјҢдјҡе°ҪеҸҜиғҪжҸҗй«ҳCPUзҡ„еҲ©з”ЁзҺҮпјҢжүҖд»ҘжӯӨж—¶еҪ“еүҚиҝӣзЁӢе°ұдјҡиў«ж“ҚдҪңзі»з»ҹеҲҮжҚўеҲ°жҢҮе®ҡи®ҫеӨҮзҡ„зӯүеҫ…йҳҹеҲ—пјҲдҫӢеҰӮй”®зӣҳпјүпјҢиҖҢCPUз»§з»ӯжү§иЎҢе…¶д»–еӯҳеңЁдәҺе°ұз»ӘйҳҹеҲ—дёӯзҡ„иҝӣзЁӢгҖӮзӯүеҫ…йҳҹеҲ—дёҺе°ұз»ӘйҳҹеҲ—з»“жһ„еҹәжң¬дёҖиҮҙпјҢд№ҹжҳҜдёҖдёӘеҸҢеҗ‘й“ҫиЎЁз»“жһ„гҖӮиҝӣзЁӢиҝӣе…Ҙзӯүеҫ…йҳҹеҲ—дёӯй“ҫжҺҘеҗҺпјҢеҜ№еә”иҝӣзЁӢзҠ¶жҖҒжӣҙж”№дёәзӯүеҫ…зҠ¶жҖҒпјҢзӯүеҫ…I/Oж“ҚдҪңе®ҢжҲҗгҖӮ

зӯүеҫ…йҳҹеҲ—е’Ңе°ұз»ӘйҳҹеҲ—зӨәж„ҸеӣҫеҰӮдёӢпјҡ

еҪ“I/Oж“ҚдҪңе®ҢжҲҗпјҢз»§з»ӯиҝӣе…Ҙе°ұз»ӘйҳҹеҲ—зӯүеҫ…иў«и°ғеәҰжү§иЎҢиҝӣе…ҘиҝҗиЎҢзҠ¶жҖҒ

з»јдёҠжүҖиҝ°пјҡзӯүеҫ…зҡ„жң¬иҙЁе°ұжҳҜиҝӣе…ҘеҜ№еә”и®ҫеӨҮзҡ„зӯүеҫ…йҳҹеҲ—иҝӣиЎҢжү§иЎҢпјҢеҸӘжҳҜдёҚдјҡжү§иЎҢеҜ№еә”зҡ„д»Јз ҒпјҢиҖҢзӯүеҫ…е’ҢиҝҗиЎҢзҡ„еҲҮжҚўе°ұжҳҜиҝӣзЁӢPCBеңЁдёҚеҗҢзҡ„еҸҢеҗ‘й“ҫиЎЁз»“жһ„дёӯиҝһжҺҘ

swapеҲҶеҢә¶

swapеҲҶеҢәд»Һеӯ—йқўж„ҸжҖқдёҠжқҘзңӢе°ұжҳҜдәӨжҚўеҲҶеҢәпјҢиҜҘеҲҶеҢәдёҖиҲ¬еӯҳеңЁдәҺзЎ¬зӣҳдёӯпјҢдё»иҰҒз”ЁдәҺеҶ…еӯҳе’ҢзЎ¬зӣҳд№Ӣй—ҙзҡ„иө„жәҗдәӨжҚўпјҢдҪҶжҳҜиҝҷз§ҚдәӨжҚўе№¶дёҚжҳҜеёёи§„жҖ§зҡ„пјҢдёҖиҲ¬еҮәзҺ°дәҺеҶ…еӯҳз©әй—ҙдёҘйҮҚдёҚи¶ізҡ„жғ…еҶө

еҪ“еҶ…еӯҳз©әй—ҙдёҘйҮҚдёҚи¶іж—¶пјҢж“ҚдҪңзі»з»ҹдёәдәҶдҝқиҜҒиҮӘиә«зҡ„иҝҗиЎҢжӯЈеёёпјҢдјҡе°ҶеҪ“еүҚжӯЈеңЁзӯүеҫ…йҳҹеҲ—зҡ„иҝӣзЁӢеҜ№еә”зҡ„д»Јз Ғе’Ңж•°жҚ®ж”ҫеҲ°зЎ¬зӣҳзҡ„swapеҲҶеҢәпјҢе°ҪеҸҜиғҪеҮҸе°‘еҶ…еӯҳз©әй—ҙзҡ„еҚ з”ЁпјҢиҝҷдёӘиҝҮзЁӢд№ҹиў«з§°дёәгҖҢжҚўеҮәгҖҚпјҢжӯӨж—¶иҝӣзЁӢзҡ„зҠ¶жҖҒд№ҹиў«з§°дёәйҳ»еЎһжҢӮиө·зҠ¶жҖҒ

иҖҢеҪ“жү§иЎҢеҲ°жҢҮе®ҡиҝӣзЁӢж—¶пјҢж“ҚдҪңзі»з»ҹдјҡйҮҚж–°е°ҶеҜ№еә”иҝӣзЁӢзҡ„д»Јз Ғе’Ңж•°жҚ®д»ҺswapеҲҶеҢәеҠ иҪҪеҲ°еҶ…еӯҳпјҢд»ҺиҖҢиҫҫеҲ°жӯЈеёёиҝҗиЎҢзҡ„зӣ®зҡ„пјҢиҝҷдёӘиҝҮзЁӢд№ҹиў«з§°дёәгҖҢжҚўе…ҘгҖҚ

ж•ҙдёӘиҝҮзЁӢдёӯзҡ„гҖҢжҚўе…ҘгҖҚе’ҢгҖҢжҚўеҮәгҖҚе®һйҷ…дёҠе°ұжҳҜеҲ©з”ЁгҖҢж—¶й—ҙжҚўз©әй—ҙгҖҚзҡ„жҖқжғіпјҢеӣ дёәswapеҲҶеҢәеңЁзЎ¬зӣҳдёҠпјҢжүҖд»ҘйҒҝе…ҚдёҚдәҶдәӨжҚўйҖҹеәҰж…ўпјҢеҰӮжһңеҮәзҺ°еӨ§йҮҸзҡ„дәӨжҚўпјҢж•ҙжңәзҡ„ж•ҲзҺҮе°ұиў«еӨ§еӨ§жӢүдҪҺ

Note

еӯӨе„ҝиҝӣзЁӢ¶

еүҚйқўжҸҗеҲ°зҡ„жҳҜеӯҗиҝӣзЁӢе…Ҳз»“жқҹпјҢзҲ¶иҝӣзЁӢжІЎжңүз»“жқҹе№¶дё”дёҚжҺҘеҸ—еӯҗиҝӣзЁӢзҡ„йҖҖеҮәдҝЎжҒҜпјҢеӯҗиҝӣзЁӢе°ұеӨ„дәҺеғөе°ёзҠ¶жҖҒпјҢеҰӮжһңеҸҚиҝҮжқҘе…Ҳз»“жқҹзҲ¶иҝӣзЁӢпјҢеҶҚз»“жқҹеӯҗиҝӣзЁӢе°ұдјҡеҮәзҺ°еӯҗиҝӣзЁӢеҸҳдёәеӯӨе„ҝиҝӣзЁӢпјҢзј–иҜ‘иҝҗиЎҢеүҚйқўзҡ„д»Јз ҒпјҢз»“жқҹеҜ№еә”зҲ¶иҝӣзЁӢз»“жһңеҰӮдёӢпјҡ

ж №жҚ®дёҠйқўзҡ„з»“жһңпјҢеҸҜд»ҘзңӢеҲ°еӯӨе„ҝиҝӣзЁӢдјҡиў«зі»з»ҹжүҳз®Ў

иҝӣзЁӢдјҳе…Ҳзә§пјҢиЎЁзӨәдјҳе…Ҳиў«CPUжү§иЎҢзҡ„иҝӣзЁӢзҡ„зӯүзә§пјҢеңЁLinuxдёӢпјҢиҝӣзЁӢдјҳе…Ҳзә§зӯүзә§и¶Ҡе°ҸпјҢдјҳе…Ҳзә§и¶Ҡй«ҳпјҢиў«дјҳе…Ҳжү§иЎҢзҡ„жҰӮзҺҮи¶ҠеӨ§
