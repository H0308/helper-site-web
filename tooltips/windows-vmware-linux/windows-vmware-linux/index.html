еңЁдёӢйқўиҝҷдёӘз•Ңйқўзӯүеҫ…дёҖж®өж—¶й—ҙпјҡ

зӮ№еҮ»з«ӢеҚійҮҚеҗҜпјҡ

жҸҗзӨәз§»йҷӨд»ӢиҙЁж—¶пјҢеңЁиҷҡжӢҹжңәз•ҢйқўзӣҙжҺҘзӮ№еҮ»EnterеҚіеҸҜпјҢжҺҘдёӢжқҘзӮ№еҮ»еҲӣе»әеҘҪзҡ„з”ЁжҲ·е№¶иҫ“е…ҘеҜҶз ҒзЎ®е®ҡпјҡ

еҪ“еүҚиҝӣе…Ҙй»ҳи®ӨжҳҜжҷ®йҖҡз”ЁжҲ·пјҢдёәдәҶдҝқиҜҒжҺҘдёӢжқҘзҡ„ж“ҚдҪңпјҢйңҖиҰҒеҲҮжҚўеҲ°rootз”ЁжҲ·пјҢй»ҳи®Өжғ…еҶөдёӢпјҢUbuntuзі»з»ҹй»ҳи®Өrootз”ЁжҲ·еҜҶз ҒжҳҜйҡҸжңәзҡ„пјҢдҪҝз”ЁдёӢйқўзҡ„жҢҮд»Өжӣҙж”№rootз”ЁжҲ·еҜҶз Ғпјҡ

Bash
1
sudo passwd

иҫ“е…ҘеҪ“еүҚз”ЁжҲ·зҡ„еҜҶз ҒпјҲй»ҳи®ӨдёҚеӣһжҳҫпјүеҗҺиҫ“е…Ҙrootз”ЁжҲ·зҡ„еҜҶз ҒпјҲдёҚиҰҒеӨӘз®ҖеҚ•пјҢдёҚз„¶дјҡжҸҗзӨәеҜҶз Ғж— ж•ҲпјүпјҢдҪҝз”Ёsu -е‘Ҫд»ӨеҲҮжҚўдёәrootз”ЁжҲ·еҰӮдёӢпјҡ

й…ҚзҪ®UbuntuзҪ‘з»ң¶

жӯҘйӘӨ 1: еңЁиҷҡжӢҹжңәдёӯе®үиЈ…е№¶й…ҚзҪ®SSHжңҚеҠЎ¶

  1. зЎ®дҝқзҪ‘з»ңиҝһжҺҘпјҡеңЁеүҚйқўVMwareзҡ„й…ҚзҪ®дёӯи®ҫзҪ®еҘҪдәҶNATеҹәжң¬е°ұжІЎжңүй—®йўҳ

  2. е®үиЈ…SSHжңҚеҠЎеҷЁпјҡеңЁиҷҡжӢҹжңәдёӯжү“ејҖз»Ҳз«ҜпјҢз„¶еҗҺе®үиЈ…OpenSSHжңҚеҠЎеҷЁпјҡ

    Bash
    1
2
    sudo apt update
sudo apt install openssh-server

  3. й…ҚзҪ®SSHпјҡй»ҳи®Өжғ…еҶөдёӢпјҢSSHеә”иҜҘе·Із»Ҹи®ҫзҪ®еҘҪе№¶зӣ‘еҗ¬еңЁз«ҜеҸЈ22дёҠпјҢй»ҳи®ӨеҸҜиғҪжҳҜжіЁйҮҠзҡ„пјҢдҪҝз”ЁдёӢйқўзҡ„е‘Ҫд»Өжү“ејҖж–Үд»¶жЈҖжҹҘдёҖдёӢпјҡ

    Bash
    1
    sudo nano /etc/ssh/sshd_config

    зЎ®дҝқPort 22иҝҷдёҖиЎҢжІЎжңүиў«жіЁйҮҠжҺүпјҲеҚіеүҚйқўжІЎжңү#пјүгҖӮеҰӮжһңдҪ жғіиҰҒдҪҝз”Ёе…¶д»–з«ҜеҸЈпјҢеҸҜд»Ҙдҝ®ж”№PortеҗҺйқўзҡ„ж•°еӯ—пјҢеҰӮдёӢеӣҫжүҖзӨәпјҡ

  4. йҮҚеҗҜSSHжңҚеҠЎпјҡжӣҙж”№еҗҺпјҢдҪҝз”ЁдёӢйқўзҡ„д»Јз ҒйҮҚеҗҜSSHжңҚеҠЎд»Ҙеә”з”Ёжӣҙж”№пјҡ

    Bash
    1
    sudo service ssh restart

  5. иҺ·еҸ–иҷҡжӢҹжңәIPең°еқҖпјҡжүҫеҲ°иҷҡжӢҹжңәзҡ„IPең°еқҖпјҢиҝҷж ·еҸҜд»Ҙд»ҺеӨ–йғЁиҝһжҺҘе®ғпјҡ

    Bash
    1
    hostname -I

жӯҘйӘӨ 2: д»Һжң¬ең°и®Ўз®—жңәиҝһжҺҘеҲ°иҷҡжӢҹжңә¶

  1. жү“ејҖз»Ҳз«ҜпјҡеңЁдҪ зҡ„жң¬ең°и®Ўз®—жңәдёҠжү“ејҖдёҖдёӘз»Ҳз«ҜзӘ—еҸЈпјҢд»ҘXshellдёәдҫӢ

  2. дҪҝз”ЁSSHиҝһжҺҘеҲ°иҷҡжӢҹжңәпјҡ

    зӮ№еҮ»ж–°е»әдјҡиҜқ

    жҢүз…§дёӢйқўеӣҫзүҮиҫ“е…ҘеҶ…е®№пјҡ

  3. иҫ“е…ҘеҜҶз Ғпјҡзі»з»ҹдјҡжҸҗзӨәиҫ“е…ҘеҜҶз ҒгҖӮиҫ“е…ҘжӯЈзЎ®зҡ„еҜҶз ҒеҗҺпјҢдҪ е°Ҷзҷ»еҪ•еҲ°иҷҡжӢҹжңә

Note

жӯӨеӨ„е»әи®®е…Ҳз”Ёжҷ®йҖҡз”ЁжҲ·зҷ»еҪ•пјҢеҶҚйҖҡиҝҮжҷ®йҖҡз”ЁжҲ·еҲҮжҚўеҲ°rootз”ЁжҲ·

rootзҷ»еҪ•¶

еҰӮжһңжғіеңЁз»Ҳз«ҜдёӯзӣҙжҺҘдҪҝз”Ёrootзҷ»еҪ•еҸҜд»ҘжҢүз…§дёӢйқўзҡ„жӯҘйӘӨи®ҫзҪ®пјҡ

  1. дҝ®ж”№SSHй…ҚзҪ®ж–Үд»¶пјҡеңЁиҷҡжӢҹжңәдёӯзј–иҫ‘sshd_configж–Үд»¶пјҢе№¶еҸ–ж¶ҲжіЁйҮҠжҲ–жӣҙж”№д»ҘдёӢиЎҢпјҡ

    Bash
    1
    PermitRootLogin yes

    еҰӮдёӢеӣҫжүҖзӨәпјҡ

  2. йҮҚеҗҜSSHжңҚеҠЎпјҡжӣҙж”№еҗҺпјҢдҪҝз”ЁдёӢйқўзҡ„д»Јз ҒйҮҚеҗҜSSHжңҚеҠЎд»Ҙеә”з”Ёжӣҙж”№пјҡ

    Bash
    1
