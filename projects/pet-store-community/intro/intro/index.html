宠物商城与社区系统¶

约 3351 个字 预计阅读时间 11 分钟

项目介绍¶

宠物商城与社区系统是一个集宠物商品交易、宠物社区互动、用户消息通知、在线客服以及后台运营管理于一体的综合性信息系统。系统面向普通用户提供商品浏览、购物下单、社区交流、账号管理等服务，同时面向管理员提供商品管理、订单管理、用户管理、社区内容审核和系统通知等后台能力

从当前项目代码结构来看，系统采用前后端分离架构，划分为用户端前台、管理员后台和后端服务三大部分。用户端主要承担面向消费者和社区用户的业务交互，管理员后台主要承担运营维护与内容治理工作，后端服务则提供统一的接口、鉴权、搜索、推荐、文件存储、消息推送与实时通信能力

技术架构¶

后端技术栈¶

  • Spring Boot：作为后端核心开发框架，用于组织各类业务模块与接口服务
  • Spring Security：用于实现用户与管理员身份认证、接口访问控制和权限校验
  • JWT Token：用于维护登录状态，实现前后端分离场景下的身份识别
  • MyBatis Plus：用于数据库访问、分页查询以及常规持久层开发
  • MySQL：作为系统主要业务数据库，存储用户、商品、订单、帖子、评论等核心数据
  • Redis：用于缓存、验证码及部分业务状态管理
  • RabbitMQ：用于注册通知、审核通知等异步消息处理
  • WebSocket：用于在线客服和消息实时通信
  • Elasticsearch：用于用户端搜索功能
  • Knife4j与SpringDoc：用于生成和维护接口文档
  • Java Mail：用于邮箱验证码和邮件通知
  • 阿里云OSS：用于商品图片、帖子媒体、评论媒体等文件上传与存储
  • 百度地图API：用于物流轨迹及地图相关信息展示
  • Spring AI Alibaba：提供AI服务支持

前端技术栈¶

用户端技术栈¶

  • Vue 3：构建用户端页面与交互逻辑。
  • Vue Router：实现前台页面路由管理
  • Element Plus：提供用户端常用UI组件
  • Sass：用于样式开发与主题组织
  • TipTap：用于帖子富文本编辑
  • html2canvas：用于页面内容转图片能力
  • Three.js：用于部分 3D 场景展示
  • marked、highlight.js、lowlight：用于 Markdown 渲染和代码高亮显示

管理端技术栈¶

  • Vue 3 + TypeScript：构建后台管理页面，提高可维护性
  • Vue Router：实现后台菜单路由管理
  • Pinia：用于后台状态管理
  • Element Plus：提供后台管理界面组件能力
  • ECharts：用于统计图表与运营数据可视化展示
  • Tailwind CSS：辅助后台样式开发
  • WangEditor：用于后台富文本编辑能力
  • Axios：用于后台接口请求

用户端功能分析¶

首页与商城入口功能¶

系统首页承担平台展示与用户引导作用，主要功能包括：

  • 展示品牌信息和系统导航入口
  • 展示轮播图内容
  • 展示最新商品与推荐内容
  • 提供进入商城、社区、个人中心等核心页面的入口

该模块主要用于提升用户对平台的第一印象，并引导用户快速进入主要业务场景

商城浏览功能¶

商城模块为用户提供商品浏览与筛选能力，主要包括：

  • 商品列表展示
  • 一级分类和二级分类浏览
  • 按分类查看商品内容
  • 商品搜索结果展示
  • 推荐商品展示

从当前代码实现来看，商城主页面、分类页面和搜索页面之间具有较强的结构复用关系，能够支持用户从不同入口进入商品浏览流程

商品详情功能¶

系统中的商品详情区分为宠物详情页与宠物用品详情页两类，主要功能包括：

  • 展示商品图片、名称、价格和描述信息
  • 展示评论内容
  • 支持用户查看商品详情后进入购买流程
  • 支持登录用户进行评论、回复评论、加入购物车和立即购买

该模块是用户完成浏览转化的重要环节，也是商品信息展示和用户交互的核心页面

购物车功能¶

购物车模块主要用于实现用户的集中式商品管理与统一结算，主要功能包括：

  • 查看购物车商品列表
  • 修改商品购买数量
  • 删除单个商品
  • 批量删除购物车商品
  • 选择商品进入结算流程
  • 支持单个商品或多个商品批量结算

未登录用户在尝试进行购物车相关操作时，系统会提示其先完成登录

订单与结算功能¶

订单模块是系统交易流程的关键组成部分，主要包含以下功能：

  • 预创建订单
  • 修改订单中的用户信息，如收货地址、联系方式等
  • 修改商品购买数量
  • 提交订单
  • 单笔订单支付与批量订单支付
  • 取消订单与批量取消订单
  • 确认收货
  • 申请退款与取消退款
  • 查询物流信息
  • 查看订单详情

该模块覆盖了从下单、支付、发货到售后退款的完整交易链路

个人中心功能¶

个人中心模块用于集中管理用户个人信息及个人业务数据，当前包括以下子模块：

  • 个人资料
  • 我的订单
  • 我的帖子
  • 收藏帖子
  • 通知箱
  • 账号设置

该模块为用户提供统一的账户管理入口，是连接交易业务与社区业务的重要枢纽

个人资料与账号管理功能¶

个人资料和账号设置相关功能包括：

  • 查看用户基础信息
  • 修改昵称、头像等个人资料
  • 进行实名认证
  • 查看和修改手机号
  • 查看和修改邮箱
  • 修改登录密码
  • 注销账号
  • 退出登录

该模块保证了用户能够对自身账户信息进行自助维护，也体现了系统对账户安全与可管理性的支持

登录与注册功能¶

当前系统支持较完整的用户认证流程，主要包括：

  • 邮箱密码登录
  • 邮箱注册
  • 图片验证码校验
  • 邮箱验证码校验
  • Google OAuth 登录
  • 登录态失效后的自动重新登录提示
  • 全局登录弹窗唤起

该模块兼顾了基础账号体系与第三方登录接入需求，提升了用户登录的灵活性和可用性

社区浏览功能¶

社区模块为用户提供内容交流与互动空间，主要功能包括：

  • 浏览帖子列表
  • 按栏目查看帖子
  • 按话题查看帖子
  • 查看推荐帖子
  • 查看帖子详情

该模块增强了平台的社交属性，有利于形成围绕宠物主题的内容社区生态

发帖与帖子编辑功能¶

系统支持用户发布和编辑帖子，主要包括：

  • 创建帖子
  • 编辑已有帖子
  • 选择栏目与话题
  • 使用富文本编辑器编写内容
  • 上传帖子媒体内容

该模块要求用户已登录，确保内容发布主体可追溯

帖子互动功能¶

帖子互动功能是社区模块的重要组成部分，主要包括：

  • 查看帖子评论列表
  • 发布帖子评论
  • 回复评论
  • 删除本人评论
  • 点赞帖子
  • 点踩帖子
  • 收藏帖子
  • 查询点赞、点踩、收藏状态

该功能模块增强了用户参与感，提高了社区内容的活跃度

收藏与我的帖子功能¶

为了便于用户沉淀内容和回访，系统提供了：

  • 我的帖子查看功能
  • 收藏帖子查看功能
  • 对个人内容和收藏内容的统一管理

这类功能可以提升用户留存率，并增强社区内容的长期价值

通知箱功能¶

通知箱模块主要用于承载系统消息与业务消息，当前主要功能包括：

  • 获取通知列表
  • 标记单条消息已读
  • 全部标记为已读
  • 查询未读数量
  • 删除通知

该模块可用于承接注册通知、审核通知、订单状态通知等多类业务消息

在线客服功能¶

系统提供用户与客服之间的在线消息交互能力，主要包括：

  • 获取客服会话消息列表
  • 查看历史消息
  • 标记消息已读
  • 上传聊天媒体
  • 获取客服信息
  • 查询未读消息状态

在线客服模块有助于提升售前和售后服务体验

AI 聊天功能¶

当前项目中还包含 AI 聊天功能，主要包括：

  • 创建AI会话
  • 获取AI会话列表
  • 查看历史对话记录
  • 修改会话标题
  • 删除会话
  • 发起AI对话请求（包括询问商品信息、商品推荐、订单查询、物流查询（工具调用）、商城帮助（RAG））

该模块为平台增加了智能辅助能力，能够在咨询、陪伴或内容辅助等场景中发挥作用

管理员后台功能分析¶

管理员登录与后台入口¶

管理员后台支持独立的认证流程，主要功能包括：

  • 管理员登录
  • 管理员退出登录
  • 登录后进入后台工作台
  • 通过权限机制控制后台页面与接口访问

后台首页与数据看板功能¶

后台工作台主要用于展示平台整体运营数据，主要包括：

  • 用户数量统计
  • 商品数量统计
  • 订单数量统计
  • 订单金额统计
  • 新用户增长情况
  • 热门话题统计
  • 库存预警信息

该模块能够帮助管理员快速掌握平台运行状况

商品管理功能¶

后台商品管理模块主要包括：

  • 查看商品列表
  • 查看宠物商品详情
  • 查看用品商品详情
  • 修改商品信息
  • 新增宠物商品
  • 新增宠物用品商品
  • 商品上架
  • 商品下架
  • 删除商品
  • 批量删除商品

该模块是后台运营管理中最核心的业务模块之一

商品分类管理功能¶

商品分类管理模块主要包括：

  • 分类搜索
  • 一级分类管理
  • 二级分类管理
  • 新增分类
  • 修改分类
  • 删除分类
  • 批量删除分类

该模块用于维护商城商品结构，为前台分类浏览提供基础数据支持

用户管理功能¶

后台用户管理模块主要包括：

  • 查看用户列表
  • 修改用户信息
  • 重置用户状态
  • 停用用户
  • 删除用户
  • 批量删除用户
  • 封禁用户
  • 解封用户
  • 批量封禁与解封

该模块为平台提供了基础的用户治理能力

社区内容管理功能¶

后台社区内容管理主要分为帖子管理、话题管理和栏目管理三部分

帖子管理¶

  • 帖子列表查询
  • 帖子详情查看
  • 删除帖子
  • 批量删除帖子
  • 帖子审核

话题管理¶

  • 话题列表查询
  • 新增话题
  • 修改话题

栏目管理¶

  • 栏目列表查询
  • 新增栏目
  • 修改栏目
  • 删除栏目
  • 批量删除栏目

上述模块共同构成后台社区内容治理体系，用于维护社区秩序与内容质量

评论管理功能¶

后台评论管理模块主要包括：

订单管理功能¶

后台订单管理模块主要包括：

  • 查询订单列表

该模块承担商城后台的订单履约和售后处理职责

客服消息处理功能¶

后台客服中心主要用于处理用户咨询消息，主要包括：

  • 查看用户最新消息
  • 查看会话历史消息
  • 标记消息已读
  • 查询用户信息

该模块与用户端在线客服功能共同组成完整的消息交互体系

敏感词与系统设置功能¶

后台系统设置模块主要包括：

  • 敏感词管理
  • 敏感词分类管理
  • 邮件通知配置页面

其中，敏感词管理功能支持新增、删除、批量导入和分类管理，可用于帖子、评论内容审核场景

系统通知功能¶

后台支持向用户发送系统通知，用于触达平台消息、业务提醒和运营通知等内容。该功能有助于增强平台管理效率与用户感知能力
